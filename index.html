Connect to Google Sheets — 3 easy steps
Your data syncs to a Google Sheet in real-time. Open FinanceDesk on your phone, tablet, or any browser and it reads from the same sheet. Uses your existing Google / Gmail account — no subscription needed.
1
Go to console.cloud.google.com → Create a new project named FinanceDesk.
Then go to APIs & Services → Library → search Google Sheets API → Enable it.
Also enable Google Drive API.
2
Go to APIs & Services → OAuth consent screen → choose External → fill App name FinanceDesk, your email → Save.
Then go to Credentials → Create Credentials → OAuth 2.0 Client ID.
Application type: Web application.
Authorised JavaScript origin: null (for local file) — or your hosting URL.
Copy the Client ID shown.
3
Paste the Client ID below and click Connect Google Account.
FinanceDesk will create a Google Sheet called FinanceDesk_Data in your Google Drive automatically.
